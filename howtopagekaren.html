    If statements tell us that if a certain condition is found to be true, then the block of code in the statement will be performed. We used if statements by putting in the variables for the answer in place of the conditions. For the code blocks, we wrote what your fortune would be for the answer you choose. For example if you were to choose apple for the first question then the output, which is the code block, would be that you are to go to Stanford University.
